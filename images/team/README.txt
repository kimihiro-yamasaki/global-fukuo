team avatars
============

現在配置されているアバター画像:

  - kimi.png
  - jaisy.png
  - ryo.png
  - rifan.png

これらは DiceBear (https://www.dicebear.com/) の "avataaars" スタイルで
生成された無料アバターです。
ライセンス: アバター本体は MIT、avataaars コレクションは CC BY 4.0
(Pablo Stanley 氏作、商用利用可)。

差し替えたい場合は、同じファイル名で画像を上書きするか、
別のファイル名にして page-about.php の $members 配列内の
'image' パスを更新してください。

推奨フォーマット: 正方形 (1:1)、512x512 以上、png または jpg。
ファイルが見つからない場合は、自動的に名前のイニシャル
(K / J / R / R) が円形アバター内にフォールバック表示されます。
